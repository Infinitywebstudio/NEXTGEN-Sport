# NEXTGEN SPORT - Dashboard React

Dashboard professionnel pour la plateforme de scouting de talents sportifs NEXTGEN SPORT.

## Technologies

- **React 19** + **TypeScript**
- **Material-UI (MUI) v7**
- **Recharts** - Graphiques
- **Vite** - Build tool

## Couleurs

- 🟠 **Primaire** : `#f87028` (Orange) - Boutons et éléments principaux
- 🟢 **Secondaire** : `#4bc3b9` (Turquoise)
- 🌿 **Tertiaire** : `#8fc92f` (Vert)

## Pages

1. **Vue d'ensemble** - Statistiques et graphiques
2. **Mon Profil** - Formulaire éditable
3. **Parcours** - Timeline des expériences
4. **Portfolio** - Vidéos et images
5. **Chat** - Messagerie
6. **Paramètres** - Configuration du compte

## Démarrage

```bash
# Installation
npm install

# Développement
npm run dev

# Build
npm run build
```

Le dashboard sera accessible sur : **http://localhost:5173/**
